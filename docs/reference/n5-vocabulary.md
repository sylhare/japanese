# N5 Vocabulary Reference

A comprehensive reference for all essential vocabulary needed for the Japanese Language Proficiency Test (JLPT) N5 level, organized by theme and type. This vocabulary list has been compiled based on commonly recognized N5-level vocabulary from multiple sources. 
([JLPT](https://www.jlpt.jp/e/) does not publish official vocabulary lists, but indicates the level of language knowledge expected)

## Basic Particles

| Hiragana | Romaji | Function | Example Meaning |
|----------|--------|----------|-----------------|
| は | wa | topic marker | as for... |
| が | ga | subject marker | (subject) does |
| を | wo/o | object marker | (direct object) |
| に | ni | direction/time/location | to, at, in, on |
| へ | e | direction | to, toward |
| で | de | location of action/means | at, in, by |
| と | to | and/with | and, with |
| も | mo | also, too | also, too |
| の | no | possessive/modifier | of, 's |
| か | ka | question marker | ? |
| ね | ne | confirmation | right?, isn't it? |
| よ | yo | emphasis | I tell you |
| から | kara | from, because | from, since, because |
| まで | made | until, to | until, to, up to |
| や | ya | and (non-exhaustive) | and, etc. |
