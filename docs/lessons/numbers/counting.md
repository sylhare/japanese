# Reading Numbers

Learn the Japanese number system and how to read large numbers.

## Basic Number System

Japanese numbers follow a consistent pattern with unique groupings.

### Basic Numbers (1-10)

| Number | Hiragana | Romaji | English |
|--------|----------|--------|---------|
| 1 | いち | ichi | one |
| 2 | に | ni | two |
| 3 | さん | san | three |
| 4 | よん / し | yon / shi | four |
| 5 | ご | go | five |
| 6 | ろく | roku | six |
| 7 | なな / しち | nana / shichi | seven |
| 8 | はち | hachi | eight |
| 9 | きゅう / く | kyuu / ku | nine |
| 10 | じゅう | juu | ten |

## Number Grouping System

**Important:** Japanese numbers are grouped by **4 digits**, not 3 like in English!

```
English:    1,000,000 (groups of 3)
Japanese: 100,0000 (groups of 4)
```

### Visual Grouping

```
                  1 0 じゅう (10)
                1 0 0 ひゃく (100)
              1 0 0 0 せん (1,000)
1 | 2 0 0 0 | 0 0 0 0 いちおく にせんまん
    おく       まん
```

## Reading Examples

### Small Numbers (1-99)

- **10** = じゅう (juu)
- **23** = にじゅう さん (nijuu san) - 2×10 + 3
- **45** = よんじゅう ご (yonjuu go) - 4×10 + 5
- **78** = ななじゅう はち (nanajuu hachi) - 7×10 + 8
- **99** = きゅうじゅう きゅう (kyuujuu kyuu) - 9×10 + 9

### Hundreds (100-999)

| Number | Hiragana | Romaji | Notes |
|--------|----------|--------|-------|
| 100 | ひゃく | hyaku | |
| 200 | にひゃく | nihyaku | |
| 300 | さんびゃく | sanbyaku | Sound change! |
| 400 | よんひゃく | yonhyaku | |
| 500 | ごひゃく | gohyaku | |
| 600 | ろっぴゃく | roppyaku | Sound change! |
| 700 | ななひゃく | nanahyaku | |
| 800 | はっぴゃく | happyaku | Sound change! |
| 900 | きゅうひゃく | kyuuhyaku | |

:::tip Sound Changes (Euphony)
Watch for sound changes at 300, 600, and 800!
- 300: さん**びゃく** (not さんひゃく)
- 600: ろっ**ぴゃく** (not ろくひゃく)
- 800: はっ**ぴゃく** (not はちひゃく)
:::

## Large Numbers

### Ten Thousands (万 - man)

In Japanese, 10,000 is a single unit called **まん** (man).

| Number | Hiragana | Romaji | English |
|--------|----------|--------|---------|
| 10,000 | いちまん | ichiman | ten thousand |
| 20,000 | にまん | niman | twenty thousand |
| 50,000 | ごまん | goman | fifty thousand |
| 100,000 | じゅうまん | juuman | one hundred thousand |

### Combining Units

**12,000,000 = 1,200万 (sen nihyaku man)**

Read as: **せん にひゃく まん**

**Example:** 20,500,000
- Group by 4: 2050 | 0000
- Read: にせん ごじゅう まん (nisen gojuu man)

## Practice Examples

### Real Numbers

- **1,234** → せん にひゃく さんじゅう よん (sen nihyaku sanjuu yon)
- **5,678** → ごせん ろっぴゃく ななじゅう はち (gosen roppyaku nanajuu hachi)
- **10,000** → いちまん (ichiman) - Grouped as 1,0000
- **12,345,678** → せん にひゃく さんじゅう よん まん ごせん ろっぴゃく ななじゅう はち
  - (sen nihyaku sanjuu yon man gosen roppyaku nanajuu hachi)
  - Grouped as 1234,5678

---

## Key Points to Remember

1. **Group by 4 digits** in Japanese (not 3 like English)
2. **まん (10,000)** is a fundamental unit in Japanese
3. **Sound changes** occur with certain numbers:
   - 300, 600, 800 (hundreds)
   - 3,000, 8,000 (thousands)
4. Use **よん** (not し) and **なな** (not しち) in most contexts
5. **いち** (1) is often dropped before じゅう, ひゃく, せん but kept with まん

## Common Mistakes

❌ **Wrong:** 10,000 = じゅう せん (juu sen)
✓ **Correct:** 10,000 = いち まん (ichi man)

❌ **Wrong:** 300 = さんひゃく
✓ **Correct:** 300 = さんびゃく

❌ **Wrong:** Grouping 1,234,567 like English (1,234,567)
✓ **Correct:** Group like Japanese (123,4567)
