# Present Negative - ない

The nai-form expresses negation — it means "not doing" something.

## Group 1 (Godan) — Change to あ-column + ない

Change the final う-sound to the あ-column equivalent, then add ない:

| Ending | Nai Form | Pattern |
|--------|----------|---------|
| く | かない | ku → ka |
| ぐ | がない | gu → ga |
| す | さない | su → sa |
| つ | たない | tsu → ta |
| ぬ | なない | nu → na |
| む | まない | mu → ma |
| る | らない | ru → ra |
| う | **わない** | u → wa |

:::caution Special Case: う-ending verbs
Verbs ending in う become **わない**, not ~~あない~~
- 買う → か**わ**ない (kawanai)
- 会う → あ**わ**ない (awanai)
:::

### Examples
- かく → かかない (don't write)
- およぐ → およがない (don't swim)
- はなす → はなさない (don't speak)
- まつ → またない (don't wait)
- よむ → よまない (don't read)
- つくる → つくらない (don't make)
- かう → かわない (don't buy)

## Group 2 (Ichidan) — Simple!

| Ending | Nai Form | Pattern |
|--------|----------|---------|
| る | ない | ru → nai |

- たべる → たべない (don't eat)
- みる → みない (don't see)
- おきる → おきない (don't wake up)
- ねる → ねない (don't sleep)

## Group 3 (Irregular)

- する → しない (don't do)
- くる → こない (don't come)

:::caution Special Case: ある
The verb ある (to exist/have) has an irregular negative form:
- ある → **ない** (not あらない)

This is the only verb where the negative form is completely different!
:::

## Nai Form in Context

**Simple negation:**
> にくを**たべない**。
> *I don't eat meat.*

**With んです for explanation:**
> にほんごが**わからない**んです。
> *I don't understand Japanese (explanation).*

**Negative request (ないでください):**
> ここでしゃしんを**とらないで**ください。
> *Please don't take photos here.*

:::info Common Nai Form Patterns
- **〜ないでください** — negative request: さわらないでください (please don't touch)
- **〜なければならない** — must do: いかなければならない (must go)
- **〜なくてもいい** — don't have to: こなくてもいい (don't have to come)
- **〜ないで** — without doing: たべないでねた (slept without eating)
- **〜ないほうがいい** — better not to: いわないほうがいい (better not say)
:::
