# Present - て form

The te-form is one of the most versatile forms in Japanese. It's used to:
- Connect multiple actions
- Make requests (〜てください)
- Express ongoing states (〜ている)
- Give and receive actions (〜てあげる/もらう)

:::tip Polite Form
For polite requests, add **ください** after the te-form:
- たべて → たべてください (please eat)
- まって → まってください (please wait)
- して → してください (please do)
:::

## Group 1 (Godan) — Pattern Based on Ending

The te-form depends on the final syllable:

| Ending | Te Form | Pattern |
|--------|---------|---------|
| く | いて | k → ite |
| ぐ | いで | g → ide |
| す | して | s → shite |
| つ | って | t → tte |
| る | って | r → tte |
| う | って | u → tte |
| ぬ | んで | n → nde |
| む | んで | m → nde |
| ぶ | んで | b → nde |

:::tip Memory Trick
**って group**: つ, る, う (sounds like "tsu-ru-u")  
**んで group**: ぬ, む, ぶ (the "nasal" consonants)
:::

:::warning Special Exception: 行く
行く (iku - to go) is irregular in te-form:  
- 行く → **いって** (itte), NOT ~~いいて~~
:::

### Examples

- かく → かいて (write)
- およぐ → およいで (swim)
- はなす → はなして (speak)
- まつ → まって (wait)
- つくる → つくって (make)
- かう → かって (buy)
- しぬ → しんで (die)
- よむ → よんで (read)
- あそぶ → あそんで (play)

## Group 2 (Ichidan) — The Easy One

| Ending | Te Form | Pattern |
|--------|---------|---------|
| る | て | ru → te |

### Examples

- たべる → たべて (eat)
- みる → みて (see)
- おきる → おきて (wake up)
- ねる → ねて (sleep)

## Group 3 (Irregular)

### Examples

- する → して (do)
- くる → きて (come)
- べんきょうする → べんきょうして (study)
- さんぽする → さんぽして (take a walk)

## Te Form in Context

**Connecting actions:**
> あさごはんを**たべて**、がっこうにいきます。
> *I eat breakfast and go to school.*

**Making requests:**
> ちょっと**まって**ください。
> *Please wait a moment.*

**Ongoing state:**
> ほんを**よんで**います。
> *I am reading a book.*

:::info Common Te Form Patterns
- **〜ている** — ongoing action/state: たべている (is eating)
- **〜てください** — polite request: みてください (please look)
- **〜てもいい** — permission: たべてもいいですか (may I eat?)
- **〜てはいけない** — prohibition: はいってはいけない (must not enter)
- **〜てあげる/もらう/くれる** — giving/receiving actions
:::
