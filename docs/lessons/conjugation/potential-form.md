# Potential Form - できる

The potential form expresses ability — "can do" or "is able to do" something.

- Express what you are able to do
- Express what is possible in a given situation
- Ask if someone is capable of something

:::tip Polite Form
For polite speech, replace the final る with ます:
- かける → かけます (can write)
- たべられる → たべられます (can eat)
- できる → できます (can do)
:::

## Group 1 (Godan) — Change to え-column + る

Change the final う-sound to its え-column equivalent, then add る:

| Ending | Potential Form | Pattern |
|--------|---------------|---------|
| く | ける | ku → keru |
| ぐ | げる | gu → geru |
| す | せる | su → seru |
| つ | てる | tsu → teru |
| る | れる | ru → reru |
| う | える | u → eru |
| ぬ | ねる | nu → neru |
| む | める | mu → meru |
| ぶ | べる | bu → beru |

:::tip Memory Trick
Every Godan verb follows the same pattern: **swap the う-sound for え, then add る**.
く → け, む → め, る → れ, and so on.
:::

### Examples

- かく → かける (can write)
- たのしむ → たのしめる (can enjoy)
- およぐ → およげる (can swim)
- はなす → はなせる (can speak)
- まつ → まてる (can wait)
- つくる → つくれる (can make)
- かう → かえる (can buy)
- あそぶ → あそべる (can play)

## Group 2 (Ichidan) — Drop る, add られる

| Ending | Potential Form | Pattern |
|--------|---------------|---------|
| る | られる | ru → rareru |

:::caution Casual shortcut: ら-drop
In casual speech, られる is often shortened to **れる** (called ら抜き言葉, ra-nuki):
- たべられる → たべ**れる** (can eat — casual)
- みられる → み**れる** (can see — casual)

Both are understood, but られる is more formal and traditionally correct.
:::

### Examples

- たべる → たべられる (can eat)
- みる → みられる (can see)
- おきる → おきられる (can wake up)
- ねる → ねられる (can sleep)
- でる → でられる (can leave)
- おぼえる → おぼえられる (can remember)
- かんがえる → かんがえられる (can think / can consider)
- おしえる → おしえられる (can teach)

## Group 3 (Irregular)

- する → **できる** (can do)
- くる → **こられる** (can come)

:::warning Special Case: する
する does not follow the え-column pattern — it becomes **できる**, a completely different word.
- べんきょうする → べんきょう**できる** (can study)
- りょうりする → りょうり**できる** (can cook)
- うんてんする → うんてん**できる** (can drive)
:::

## Potential Form in Context

**Ability statement:**
> かんじが**かける**。
> *I can write kanji.*

**Polite ability:**
> にほんごが**はなせます**か？
> *Can you speak Japanese?*

**Situation-based possibility:**
> きょうは、いっしょに**たべられる**？
> *Can we eat together today?*

**できる with noun:**
> にほんごが**できます**。
> *I can (do) Japanese. / I speak Japanese.*

:::info Common Potential Form Patterns
- **〜ことができる** — formal potential: かんじをかくことができる (I am able to write kanji)
- **〜られない** — cannot: たべられない (can't eat)
- **〜できるだけ** — as much as possible: できるだけはやく (as fast as possible)
- **〜ようになる** — to become able to: はなせるようになった (became able to speak)
- **〜なくなる** — to become unable to: ねられなくなった (became unable to sleep)
:::
